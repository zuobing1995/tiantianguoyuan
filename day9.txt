传递方式:
位置传参 
  序列传参
关键字传参
  字典关键字传参

1 位置传参:
实际传参:
  实参的对应关系与形参的对应关系是按位置来对应的

实例:
def mymin(a,b,c):
    pass
mymin(1,2,3)
说明:
实参和形参通过位置进行传递和匹配
实参的个数必须和形参的相同

2 序列传参 *
序列传参是指在函数的调用时,用'*'将序列解开,再按位置传参

说明:
序列传参时,序列拆解的位置与形参对应
序列的位置对应相应的参数位置

3 关键字传参(不按照位置传参,是按形参的关键字传参)
 关键字传参是指传参时,按着形参的名字给形参赋值  实参和形参按名字进行匹配

def myfun(a,b,c):

myfun(b=1,c=2,a=3)
4 字典关键字传参

是指实参为字典,将字典用**拆解后进行关键字传参的方式
说明:字典的键名和形参名一样
    字典的键名必须是字符串(为标识符的命名方式)
    字典的键名要在形参中存在
5 函数的综合传参
   函数的传参方式,在能确定形参能唯一匹配到相应实参的情况下可以任意组合
   函数的位置传参要先于关键字传参

   说明:位置传参和序列传参可以混合使用 没有顺序先后
   myfun(100,*[200,300])
   myfun(*[100,200],300)

   函数的关键字传参和字典关键字传参可以混合使用,且可以顺序颠倒
   myfun(c=300,**{'b':100,'a':200})
   myfun(**{'a':100,'b':200},c=300)

   位置传参和关键字传参可以混合使用,但是位置传参要先于关键字传参

   myfun(100,b=200,c=300)
   myfun(100,**{'b':200,'c':300})
   



  如何定义形参
   函数的缺省参数
      语法:
         def 函数名(形参名1=默认参数1,形参2=默认参数2,....)
         语句块

         当实参给了参数,形参就用否则就用默认参数,默认参数要写在所有的位置参数后面



函数形参的定义方式:
  位置形参
  星号元组形参
  命名关键字形参
  **字典形参


 *元组形参

  def 函数名(*元组形参名)
    语句块

作用:收集多余的位置形参

说明:元组形参名一般为args

命名关键字形参:
语法:

 def 函数名(*,命名关键字形参)
    语句块

作用:强制所有的参数都必须用关键字传参或字典关键字传参

实例:
   *后面的形参,实参必须是关键字传参,*是分隔符,*前面的位置传参可以不用关键字传参


   def fun(a,b,*args,c,d)  #c,d的实参必须是关键字命名
   def fun(a,b,*,c,d)   * *后面的形参,实参必须是关键字传参,*是分隔符,*前面的位置传参可以不用关键字传参

   双星号字典形参  将关键字传参转化为字典

   语法: def 函数名(**字典形参名)

   作用:收集多余的关键字传参
   说明:
      字典性参名一般命名为kwargs

      实参必须是关键字传参或者字典传参



函数的参数说明:
位置形参,缺省参数,星号元组形参,双星号字典形参等可以混合使用


函数参数自左到右的顺序为:
1.位置形参
2.星号元组形参
3.命名关键字形参
4.双星号字典形参
